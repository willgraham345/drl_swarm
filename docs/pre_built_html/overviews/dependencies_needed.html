Dependencies for the project

  1. Installation of Nav2: https://navigation.ros.org/build_instructions/index.html

    Nav2 Base Page: https://navigation.ros.org/index.html Nav2 Plugin Tutorial: https://navigation.ros.org/plugin_tutorials/index.html

  1. Installation of ROS 2 Galactic: https://docs.ros.org/en/galactic/Installation.html

  2. Installation of ROS 2 Humble: https://docs.ros.org/en/humble/Installation.html

  3. Crazyflie Firmware: https://github.com/bitcraze/crazyflie-firmware

  4. Crazyradio Firmware: https://github.com/bitcraze/crazyradio-firmware

  5. Crazyflie Python Client: https://github.com/bitcraze/crazyflie-clients-python

  6. Crazyflie Python Library: https://github.com/bitcraze/crazyflie-lib-python

  7. WeBots Installation: https://www.cyberbotics.com/doc/guide/installation-procedure

Install using “Installing the Debian Package with the Advanced Packaging Tool (APT)”. Not Snap!
