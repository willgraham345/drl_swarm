class drl_pkg.crazyflie_publisher.CrazyfliePublisher[source]

Publisher node for controlling the Crazyflie drone.

This node initializes the necessary publishers, subscribers, and timers for controlling the Crazyflie drone. It also handles the connection to the Crazyflie and sends flight commands if in flight mode.

Parameters:

  • URI (-) – The URI of the Crazyflie.

  • fly (-) – Determines if the Crazyflie is in testing mode.

  • config_file (-) – The path to the configuration file.

  • lh0_pose_frame (-) – The frame ID for the pose of the first lighthouse.

  • lh1_pose_frame (-) – The frame ID for the pose of the second lighthouse.
